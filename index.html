Privacy Policy

1. Data Collection & Use

  • Process translation inputs in real-time (not stored)
  • Essential device information collected:
    • Device model & OS version
    • Network connection type
    • App usage statistics
  • Subscription data managed through Google Play

2. Advertising

  • Google AdMob integration for non-intrusive ads
  • Collected ad data includes:
    • Advertising ID
    • Anonymized IP address
    • Click-through rates
  • Opt-out available in device settings

3. Subscriptions

  • Auto-renewing subscriptions via Google Play
  • Manage/cancel through Play Store account
  • No personal payment data stored

4. Data Security

  • End-to-end encryption for translations
  • Regular security audits
  • Limited data retention (max 18 months)

5. User Rights

  • Request data access/deletion
  • Opt-out of analytics tracking
  • Withdraw consent via email
